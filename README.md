# Manipulando-DOM-js
Trabalho com manipulação DOM JAVASCRIPT onde, tenho uma caixa de texto (input) para escrever o texto que quiser e ao apertar o botão de enviar, o texto aparecerá abaixo salvo. Assim vale para vários textos e irão se organizando um abaixo do outro. Adicionei um botão que limpa todos os texto já escrito e enviados e também um alerta caixa vazia.
